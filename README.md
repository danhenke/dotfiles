# Bootstrap a new Mac

Run the following to bootstrap a new Mac:

```bash
xcode-select --install
git clone https://github.com/danhenke/macos.git
./macos/install.sh
```

Install the following manually via Mac App Store.

- 1Password 7
- Numbers
- Pages
- Slack
